# Citroën Center — сайт автосервиса (citroencenter.ru)

Готовый статический сайт специализированного автосервиса по обслуживанию и ремонту
автомобилей Citroën. Сделан по техническому заданию: чистый HTML/CSS/JS, без бэкенда,
базы данных и сборщиков. Полностью адаптивный, работает сразу после копирования на хостинг.

---

## 📁 Структура проекта

```
citroencenter.ru/
├── index.html              # Главная (и единственная) страница сайта
├── assets/
│   ├── css/
│   │   └── style.css       # Все стили
│   ├── js/
│   │   └── main.js         # Интерактив: меню, форма, FAQ, фильтр цен, анимации
│   └── img/
│       └── favicon.svg     # Иконка сайта
└── README.md               # Этот файл
```

---

## 🚀 Как разместить на хостинге (инструкция)

Скопируйте всю папку `citroencenter.ru` (файл `index.html` и папку `assets`) в корневую
директорию вашего хостинга — обычно это `public_html`, `www` или `htdocs`. Файл
`index.html` должен лежать в корне, а папка `assets` — рядом с ним. После загрузки сайт
сразу откроется по адресу домена без какой-либо настройки сервера, сборки или установки
зависимостей. Никаких баз данных и серверных языков не требуется.

> Локально сайт также можно открыть, просто дважды кликнув по `index.html` в браузере.

## ✨ Что есть на сайте

- **Главный экран** с чётким позиционированием (профильный сервис Citroën) и карточкой онлайн-записи.
- **Услуги** — 10 услуг в виде карточек с номерами и иконками.
- **Преимущества** — современная bento-сетка (6 блоков).
- **Модели Citroën**, которые обслуживаются.
- **Цены** — прайс с фильтром по категориям (ТО, двигатель, подвеска, тормоза, электрика, кузов).
- **Этапы работы** — вертикальный таймлайн из 6 шагов.
- **Отзывы** клиентов.
- **Форма записи** с валидацией и сообщением об успешной отправке.
- **FAQ** — аккордеон с частыми вопросами.
- **Контакты** со схемой проезда (плейсхолдер) и всеми данными.
- **Адаптив**, мобильное меню, плавающая панель действий (Позвонить / Записаться), кнопка «наверх».

Все контактные данные (телефон, адрес, e-mail, мессенджеры, реквизиты) намеренно
заменены заглушками **ХХХХХ** согласно ТЗ — их нужно заменить на реальные перед публикацией.

---

## 🔧 Что заменить перед запуском

1. Контактные заглушки `ХХХХХ` → реальные телефон, адрес, e-mail, мессенджеры, реквизиты
   (в `index.html`, включая блок Schema.org в `<head>` и значения `tel:`/`mailto:`).
2. При необходимости — реальную карту (Яндекс.Карты / Google Maps) вместо плейсхолдера.
3. Цены в прайсе — на актуальные.
4. Форма записи имитирует отправку на стороне браузера. Чтобы заявки приходили на почту/в
   CRM, подключите сторонний обработчик форм (например, Formspree).

## 🛠 Технологии

- Семантический **HTML5**, разметка **Schema.org** (AutoRepair) для SEO.
- **CSS3**: кастомные свойства, Grid/Flexbox, bento-сетка, адаптивные сетки, анимации,
  `prefers-reduced-motion`.
- **Ванильный JavaScript** (без библиотек): фильтр прайса по категориям, валидация форм
  с доступностью (aria-invalid, role=alert), маска телефона, мобильное меню с управлением
  фокусом (focus-trap, inert), аккордеон FAQ, анимации по скроллу.
- Шрифты **Unbounded** и **Manrope** (с кириллицей) подключены через Google Fonts (CDN).
  Есть системные фолбэки — сайт остаётся читаемым даже без доступа к CDN.

Внешних зависимостей, требующих сборки или установки, **нет**.
